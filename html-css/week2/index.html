Since the commercialization of the web development has been a growing industry. The growth of this industry is being drikven by businesses wishing to use their website to sell products and services to customers [1].

Knowledge of Hyper Text Markup Language (HTML) or of programming languages is still required to use such software, but the basics can be learned and implemented quickly with the help of help files, technical books, internet tutorials, or face-to-face training.


An ever growing set of tools and techologies have helped developers build more dynamic and interactive web sites. Further, web developers now help to deliver applications as web services which were traditionally only available as applications on a desktop based computers. This has allowed for many opportunities to decentralize information and media distribution. Examples can be seen with the rise of cloud services such as Adobe Creative Cloud, Dropbox, and Google Docx. These web services allows users to interact with applications from many locations, instrad of being tied to a specific workstation for their application environment.
